# NailSpaceSoari 作業記録（2026-07-03 完了時点）

## ✅ 稼働中：サロン管理アプリ（Streamlit Cloud・GitHub main に反映済み）

構成（st.navigation 方式・ファイル名はASCII、表示は日本語）：
- `app.py` … 入口（ログイン＋ページ切り替え）
- `views/home.py` … 🏠 今日（今日の売上/件数、今日の予約＋状態変更、クイック売上入力＝メニュー2・割引つき）
- `views/yoyaku.py` … 📅 予約とカルテ（月間カレンダー、1日タイムライン 9:00-19:00、予約追加＝メニュー2/所要時間/かぶり警告、顧客カルテ）
- `views/menu_kanri.py` … 💅 メニュー管理（名前/カテゴリ/料金/所要時間/有効を表で編集）
- `views/summary.py` … 📈 月次サマリー（前月比つき6指標＋グラフ）
- `views/sales.py` … ✍️ 売上フル機能（旧app.py。メニュー2の「同時施術」→「ハンドフット」変更済み）
- `common.py` … 共通（シート接続・ログイン・営業時間 OPEN_HOUR/CLOSE_HOUR・DEFAULT_DURATION=90分）

設計：
- データ保存先は既存Googleスプレッドシート（「顧客」「予約」「メニュー」シート自動作成）
- 顧客の連絡先は保存しない方針（呼び名＋目印メモ。連絡はLINE公式で）
- st.secrets に app_password を設定するとパスワードログイン有効

## 📖 運用メモ（更新のしかた）
1. Claudeに変更を依頼 → ZIPを受け取る
2. ZIPを展開 → GitHub「Add file → Upload files」で中身を全部ドラッグ → Commit
3. 反映されない時は Streamlit の Manage app → ⋮ → Reboot app
※ Claude側からGitHubへの直接書き込みは不可（読み取り専用）だった
