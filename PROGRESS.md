# NailSpaceSoari 作業記録（2026-07-03 時点）

## このブランチ（claude/nifty-faraday-4qrs43）でやったこと

### 1. サロン管理アプリの拡張（Streamlit マルチページ化）
構成：
- `app.py` … 🏠 **今日の画面**（開くと最初に出る。今日の売上・件数、今日の予約一覧＋状態変更、クイック売上入力）
- `pages/1_📅_予約とカルテ.py` … 月間カレンダー、**1日タイムライン**（営業時間9:00〜19:00、予約が所要時間ぶんのブロックで表示、空き時間が見える）、予約追加（時間帯かぶり・営業時間オーバー警告）、顧客カルテ（登録・検索・編集・来店履歴・21日で再来アラート）
- `pages/2_💅_メニュー管理.py` … メニュー・カテゴリ・料金・**所要時間（分）**・有効/無効を表で編集
- `pages/3_📈_月次サマリー.py` … 売上合計・件数・客単価・新規・再来・再来率（前月比つき）、メニュー別/決済別/6ヶ月推移グラフ
- `pages/4_✍️_売上.py` … 従来の売上アプリ（旧 app.py、中身は無変更）
- `common.py` … 共通処理（シート接続・ログイン・スタイル・営業時間の設定 OPEN_HOUR/CLOSE_HOUR/DEFAULT_DURATION）

設計方針：
- データは既存の Google スプレッドシート（シート「顧客」「予約」「メニュー」を自動作成）
- **連絡先は保存しない**（呼び名＋目印メモで管理。連絡は LINE 公式アカウントで、が方針）
- `st.secrets` に `app_password` を入れるとパスワードログインが有効になる

### 2. ホームページ関連（website/）
- `website/index.html` … 非公開の下書きページ（noindex）
- `website/copy_draft.md` … 原稿まとめ（キャッチコピー候補・プロフィール・メニュー・スクール・お客様の声）
- 実際のHPは Claude Design で制作中（https://claude.ai/design/ のプロジェクト）
- キャッチコピーは「速さ重視のサロンではない」方向で選定中。有力候補：
  - 急がない、丁寧な施術を。
  - 回転率より、あなたの爪を。
  - きれいは、急いでつくれない。
